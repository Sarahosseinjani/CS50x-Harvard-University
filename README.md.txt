About the website

This is my final project for Harvard CS50x course. It's a website that was created in order to give everyone a chance to get driving school licence easily.
There are different sections on the website. You can see some information about courses and instructors and quizes in their pages. 
You can create account in order to register in driving institute or take quize.

Description

Web application is based on Flask framework. I used cs50 python lib for working with database.

How to use

To run the web application use these commands:
$ export FLASK_APP=application.py
$ flask run

Requirements
python 3
flask
cs50
